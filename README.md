# 🛒 Laravel Multi-Vendor E-Commerce Website

A full-featured multi-vendor e-commerce platform built with **Laravel and React**.  
It supports vendor management, Stripe payments, product variations, and order handling.

---

## 🚀 Features

- Multi-vendor system
- Vendor dashboard
- Product management with variations
- Order management
- Stripe payment integration
- Authentication system
- Background job processing (queues)
- File storage system

---

## 🧰 Requirements

Make sure your system has:

- PHP >= 8.2  
- Composer (latest version)  
- Node.js >= 18.x  
- NPM  
- MySQL >= 8.x  
- Git  
- Stripe account (for payments)
